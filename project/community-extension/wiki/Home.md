# Welcome!
***Home of the first ever modding API expansion for Civ VI!*** 🤯

Civ V has the [Community Patch](https://github.com/LoneGazebo/Community-Patch-DLL/), and now we've got this. This project is maintained by [WildW](https://github.com/Wild-W).

## Features
This project is only in its infancy, but it's still mighty powerful!
* Lua (HavokScript) extensions
  * Direct [memory manipulation](/Wild-W/CivilizationVI_CommunityExtension/wiki/Memory-Manipulation)
  * New methods and objects (Check the sidebar)
  * Dynamic [event systems](/Wild-W/CivilizationVI_CommunityExtension/wiki/Events-&-Processors)
* Database extensions
  * Currently unsupported, but planned!

## Getting Started

Tell your users to download this mod from the github or steam workshop post. As long as this mod is active and the player is in a game, API changes are in effect. This mod is as simple as a dependency to add as any other utility mod.

A full guide and tutorial will be available soon.
